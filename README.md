About the Project:
GymTune is a simple web app that helps people stay consistent in the gym.

Users can:

1.Track their workouts
2.Listen to workout music 
3.See fitness progress and analytics 
4.Download workout reports

This project combines Web Development + Data Analytics in one app.

Why I Built This>>>

Many people join the gym but quit after a few weeks.
GymTune helps users stay motivated by showing their progress and workout behaviour.

 Main Features->>
a. User Account
Register and Login system
Secure session handling

b.Workout Tracking
Start and Stop workout
Automatically calculates calories burned
Saves workout history

c. Workout Music Player
Built-in playlist
Next / Previous / Pause buttons
Songs play continuously while working out

d.Fitness Dashboard
Shows important insights like:
Total workouts done
Calories burned 
Workout streak 
Weekly goal progress 
Quit risk prediction 
Most active workout day 
Consistency score 

Data Visualization>>>
Charts show workout activity over time.

Download Report:
Users can download workout data as an Excel file.

Technologies Used:::
Python Flask (Backend)
HTML, CSS, JavaScript (Frontend)
SQLite (Database)
Chart.js (Charts)
OpenPyXL (Excel report)


***How to Run the Project
